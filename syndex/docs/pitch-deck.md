# Slide 6: AI Technology

## Proprietary Matching Algorithm

### 50+ Factors Analyzed

| Category | Factors |
|----------|---------|
| **Historical** | Past participation, role frequency, commitment sizes |
| **Sector** | Industry expertise, sub-sector specialization |
| **Geographic** | HQ location, regional focus, cross-border appetite |
| **Deal Fit** | Size range, tenor preference, pricing tolerance |
| **Relationships** | Co-lending patterns, borrower history |

### Tech Stack

- **AI Model**: Cerebras Qwen (235B parameters)
- **Training Data**: 500+ historical deals, 50+ banks
- **Accuracy**: 98% match prediction rate
- **Speed**: Results in < 5 seconds
